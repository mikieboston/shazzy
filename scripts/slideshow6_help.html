Thank you for downloading: Slideshow 6 - XML

Included are:

  • MX2004 Flash File
  • v8 Flash File

 

INSTRUCTIONS:

In order to add more effects, simply duplicate one of the effects in the library. Change it to suit your needs and don't forget to link it. (right click - linkage) Next, add it in the XML file and there you go. ;)

To use any of the effects provided, simply select them in the XML file.

 

If you want to change the size of the swf file, change the stage size and after that make these changes in the actions.as file:

 

var stagew:Number = 590; (write the new size)
var stageh:Number = 300; (write the new size)

 

That's it. :)

 

Good Luck using the file in your projects!
